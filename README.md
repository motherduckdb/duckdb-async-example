# duckdb-async-example

Standalone tiny example of using duckdb-async

To try it out:

```sh
$ npm install
$ npm run build
$ node ./dist/basic.js
```

which should produce the output:

```
[
  { range: 1 },
  { range: 2 },
  { range: 3 },
  { range: 4 },
  { range: 5 },
  { range: 6 },
  { range: 7 },
  { range: 8 },
  { range: 9 }
]
```
